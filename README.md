## Usage

Install via `npm`/`yarn`/`pnpm`:

```sh
$ yarn add -D @felix-kaestner/eslint-config
```

Add this into your `.eslintrc.json` (or other `eslint` config file):

```JavaScript
{
  "extends": ["@felix-kaestner/eslint-config"]
}
```
